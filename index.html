About Me.

A passion for performance.

I am a Full Stack Developer with 2 years of experience building scalable, data-intensive web applications using Python and Django. I am highly skilled in designing REST APIs and real-time services with Django REST Framework and WebSockets.

My expertise extends to managing asynchronous workflows with Celery and RabbitMQ, as well as optimizing PostgreSQL and Redis for performance. I have strong experience in end-to-end ownership of backend systems, from architecture to deployment, with a solid foundation in secure API design and database optimization.

System Design.

How I architect modern applications.

Every system I build follows a clean separation of concerns — users interact with a polished frontend, which communicates through a secure API Gateway to a constellation of microservices, each backed by optimized databases and caching layers. Data flows continuously, securely, and at scale.
